Gameplay Basics for Version 2

  • The game starts with the AI.
  • The AI shows you how many stone are left in each heap we have.
  • You can split one number of stone from one heap to two values only when it is your turn.
  • If there is one stone left in each heap in game so you cannot split, then you lose.
  • As long as there is more than one stone in the heap , you are safe.
  • You will win if it is the artificial intelligence's turn and there is only one stone left in each heap . he lose and you win.
